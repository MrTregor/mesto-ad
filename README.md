# Проект: Mesto

Интерактивная веб-страница для добавления фотографий, удаления фотографий и постановки лайков.

## Ссылка на проект

**Опубликованная версия проекта:** https://mrtregor.github.io/mesto-ad/

## Технологии

* HTML5
* CSS3
* JavaScript (ES6+)
* Vite (сборщик модулей)
* GitHub Pages (хостинг)

## Функциональность

* Редактирование профиля пользователя
* Добавление новых карточек с фотографиями
* Удаление карточек
* Постановка и снятие лайков
* Открытие изображений в полноэкранном режиме

## Установка и запуск

1. Клонируйте репозиторий:
```bash
https://github.com/MrTregor/mesto-ad.git
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите локальный сервер разработки:
```bash
npm run dev
```

4. Откройте приложение в браузере по адресу: http://localhost:3000

## Сборка проекта

Для создания production-сборки выполните:
```bash
npm run build
```

Результат сборки будет находиться в папке `dist`.

## Публикация проекта

Для публикации проекта на GitHub Pages выполните:
```bash
npm run deploy
```

Эта команда автоматически:
1. Соберет проект (`npm run build`)
2. Опубликует содержимое папки `dist` на GitHub Pages

После публикации проект будет доступен по адресу: https://mrtregor.github.io/mesto-ad/

## Структура проекта

